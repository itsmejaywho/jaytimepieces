# JayTimepieces — Project Init

## Overview
**JayTimepieces (jtimepieces)** — Luxury watch ecommerce marketplace inspired by Chrono24. Connects buyers with 25,000+ trusted sellers worldwide. Currently in early development (v0.0.0) with landing page complete.

## Tech Stack
| Layer       | Technology              | Version  |
|-------------|------------------------|----------|
| Runtime     | React                  | 19.2.4   |
| Routing     | React Router DOM       | 7.14.0   |
| Build       | Vite                   | 8.0.4    |
| Compiler    | Babel + React Compiler | 7.29.0 / 1.0.0 |
| Linting     | ESLint                 | 9.39.4   |
| Deployment  | Vercel (SPA rewrite)   | —        |
| Language    | JavaScript/JSX         | No TypeScript |
| CSS         | Plain CSS (co-located) | No modules, no CSS-in-JS |

## Design System

### Color Palette (CSS Variables in index.css)
| Variable        | Hex       | Usage                        |
|-----------------|-----------|------------------------------|
| `--primary`     | #1a1a2e   | Dark navy headings, primary  |
| `--secondary`   | #16213e   | Deep navy variant            |
| `--accent`      | #c9a96e   | Gold — hover, brand accent   |
| `--text`        | #333      | Body text                    |
| `--text-light`  | #666      | Secondary/muted text         |
| `--bg`          | #fff      | Page background              |
| `--bg-light`    | #f8f8f8   | Subtle background contrast   |

### Typography
- **Primary font**: `'Google Sans Flex', 'Segoe UI', system-ui, sans-serif`
- **Hero heading**: Georgia/Times (serif, italic)
- **Hero**: 2.8rem → 1.4rem (mobile)
- **Section headings**: 1.7–1.8rem, weight 600–700
- **Body**: 1rem, weight 400

### Responsive Breakpoints
| Breakpoint | Target        |
|------------|---------------|
| 1024px     | Large tablet  |
| 768px      | Tablet        |
| 480px      | Mobile        |

### Interactive States
- **Hover**: color → `#c9a96e` (gold), scale 1.1x (images)
- **Transitions**: 0.2s color, 1.5s image scale, 0.3s menu/burger

### Borders & Radius
- Cards: 6–12px radius
- Avatars: 50% (circles)
- Protection container: 20px radius

## Conventions

### Naming
- **Components**: PascalCase — `Navbar.jsx`, `HeroSection.jsx`
- **Styles**: Match component — `Navbar.css`, `HeroSection.css`
- **Images**: kebab-case — `buyer-rating.png`, `rolex1.webp`
- **Barrel exports**: `index.js` in each directory

### Import Pattern
```jsx
// From barrel exports
import { Navbar, Footer } from './components'
import { Home, Shop } from './pages'

// Direct asset imports
import logo from '../../assets/images/logo.png'
```

### CSS Pattern
- Co-located: each component has its own `.css` file
- Each component owns its own media queries
- Global resets + variables in `src/index.css`
- No utility classes — component-scoped styles only

## Authentication (UI Only — No Backend Yet)
- Login and Signup pages are visual placeholders
- Shop page uses `sessionStorage.getItem('user')` for logout
- Future: Supabase + AuthContext in `src/context/AuthContext.jsx`
